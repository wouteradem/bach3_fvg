# bach3_fvg

To run the program, just hit:

```python __main__.py```

In case you just want to calculate an orbit given a certain r_peri and r_apo, then confirm with ```y```.
To change the values of r_peri and r_apo, you need to currently change them in the ```__main__.py```
file itself.
In case  you want to run the full Schwarschild method confirm with ```n```.
To change the values of r_peri and r_apo, you need to currently change them in the ```__main__.py```
file itself.
To plot the graphs you need to uncomment the line ```plt.show```

That's it!
